# Contributing to Match-3 Puzzle Helper

First off, thank you for considering contributing to this utility tool! We want to make managing grid simulations as accurate as possible.

## How Can I Contribute?

### Reporting Bugs
* Check the current Issues list to ensure the bug hasn't already been reported.
* If you find a new bug, please open a new issue using the provided Bug Report template.

### Suggesting Enhancements
* Open an issue describing the feature you would like to see added to the matrix engine (e.g., adding support for vertical match detections).
* Explain why this feature would be useful for optimizing puzzle board scores.

### Pull Requests
1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, please include basic simulation logs.
3. Ensure your JavaScript formatting matches the existing code structure in `index.js`.
4. Submit your pull request with a clear description of the modifications.
